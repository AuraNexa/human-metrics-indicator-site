# 人类指标资料库

静态资料网页：面向公开查阅者，把修行、婚恋、孩子教育、职场用人和个人状态拆成可查阅、可理解、可复看的指标目录。

## 页面

- `index.html`：首页
- `library.html`：指标库
- `catalog.html`：全部指标目录
- `indicator.html`：指标详情
- `notice.html`：阅读须知
- `custom.html` / `sample.html`：旧入口合并提示
- `404.html`：未找到页面
- `robots.txt` / `sitemap.xml`：搜索引擎入口

## Cloudflare Pages

- 构建命令：留空
- 输出目录：`/`
- 入口文件：`index.html`
